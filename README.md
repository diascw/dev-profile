# Profile 


## Objetivo do projeto
Este é um projeto React que contém componentes para renderizar um perfil.

## Instalação
* Clone o repositório do GitHub:
``` git clone https://github.com/diascw/dev-profile.git ```

* Instale as dependências:
``` npm install ```
* Execute o projeto:
``` npm run dev ```
